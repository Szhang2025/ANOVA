The Core Idea

ANOVA tests whether the means of three or more groups are equal. It works by partitioning total variance into components: variance between groups (signal) vs. variance within groups (noise). If between-group variance is large relative to within-group variance, we reject H₀.

Hypotheses

H₀: μ₁ = μ₂ = … = μₖ (all means equal) H₁: at least one μᵢ ≠ μⱼ — not "which" — just "at least one does" Follow-up: post-hoc tests identify which pairs

Variance Partitioning

SStotal = Σij (xij − x..)² SSbetween = Σk nk (xk. − x..)² SSwithin = Σk Σi (xik − xk.)² SStotal = SSbetween + SSwithin where x.. = grand mean, xk. = group k mean

The F-Statistic

MS_between = SS_between / (K−1) MS_within = SS_within / (N−K) F = MS_between / MS_within Under H₀, F ~ F(K−1, N−K) distribution
One-Way vs Two-Way ANOVA

One-Way ANOVA

One categorical independent variable (factor) with K levels. Tests whether the group means differ. Example: do three teaching methods (A, B, C) produce different exam scores?

Source df SS MS F Between groups K−1 SS_B SS_B/(K−1) MS_B/MS_W Within groups N−K SS_W SS_W/(N−K) Total N−1 SS_T

Two-Way ANOVA

Two categorical factors. Tests three things: main effect of Factor A, main effect of Factor B, and their interaction (A×B). Requires at least one observation per cell.

Source df MS F Factor A a−1 SS_A/(a−1) MS_A/MS_W Factor B b−1 SS_B/(b−1) MS_B/MS_W A×B (a−1)(b−1) SS_AB/df MS_AB/MS_W Within ab(n−1) SS_W/df Total abn−1

Assumptions

Normality: residuals are normally distributed. ANOVA is robust to mild violations with large n.

Homogeneity of variance: all groups have equal population variance. Test with Levene's test.

Independence: observations are independent.

Effect Size

η² (eta-squared) = SS_between / SS_total — proportion of variance explained ω² (omega-squared) ← less biased

Step 1 — SStotal

Every observation's squared deviation from the grand mean (8.250):

Method A (4 − 8.25)² = (−4.25)² = 18.0625 (6 − 8.25)² = (−2.25)² = 5.0625 (8 − 8.25)² = (−0.25)² = 0.0625 subtotal = 23.1875 Method B (5 − 8.25)² = (−3.25)² = 10.5625 (7 − 8.25)² = (−1.25)² = 1.5625 ×3 (9 − 8.25)² = ( 0.75)² = 0.5625 subtotal = 15.8125 Method C (11 − 8.25)² = ( 2.75)² = 7.5625 (13 − 8.25)² = ( 4.75)² = 22.5625 (12 − 8.25)² = ( 3.75)² = 14.0625 (10 − 8.25)² = ( 1.75)² = 3.0625 subtotal = 47.2500 SStotal = 23.1875 + 15.8125 + 47.25 = 86.250 dfT = N − 1 = 12 − 1 = 11

Step 2 — SSbetween

Each group contributes nk × (group mean − grand mean)². The different nk values are what makes this design unbalanced:

Method A nA=3, xA.=6.000 3 × (6.000 − 8.250)² = 3 × (−2.250)² = 3 × 5.0625 = 15.1875 Method B nB=5, xB.=7.000 5 × (7.000 − 8.250)² = 5 × (−1.250)² = 5 × 1.5625 = 7.8125 Method C nC=4, xC.=11.500 4 × (11.500 − 8.250)² = 4 × (3.250)² = 4 × 10.5625 = 42.2500 SSbetween = 15.1875 + 7.8125 + 42.25 = 65.250 dfB = K − 1 = 3 − 1 = 2

Step 3 — SSwithin

Within each group, squared deviations from the group mean. dfW = Σ(nk − 1) = 2 + 4 + 3 = 9:

Method A (xA. = 6.000) df = nA−1 = 2 (4−6)² = 4 (6−6)² = 0 (8−6)² = 4 SSW,A = 8.000 Method B (xB. = 7.000) df = nB−1 = 4 (5−7)² = 4 (7−7)² = 0 ×3 (9−7)² = 4 SSW,B = 8.000 Method C (xC. = 11.500) df = nC−1 = 3 (11−11.5)² = 0.25 (13−11.5)² = 2.25 (12−11.5)² = 0.25 (10−11.5)² = 2.25 SSW,C = 5.000 SSwithin = 8.000 + 8.000 + 5.000 = 21.000 dfW = N − K = 12 − 3 = 9

Step 4 — ANOVA Table

Verify partition: SSB + SSW = 65.250 + 21.000 = 86.250 = SST ✓

Source SS df MS F p Between 65.250 2 32.625 13.982 .002 Within 21.000 9 2.333 Total 86.250 11 MSB = SSB / dfB = 65.250 / 2 = 32.625 MSW = SSW / dfW = 21.000 / 9 = 2.333 F = MSB / MSW = 32.625 / 2.333 = 13.982 η² = SSB / SST = 65.250 / 86.250 = 0.757 (large) ω² = (SSB − dfB·MSW) / (SST + MSW) = (65.250 − 2×2.333) / (86.250 + 2.333) = 0.684
The fundamental identity — holds for every observation xij regardless of group size:
(xij − x..) = (xk. − x..) + (xij − xk.) total deviation = between-group part + within-group part
Concrete check — observation x = 13 in Method C (xC. = 11.500, x.. = 8.250):
(13 − 8.250) = (11.500 − 8.250) + (13 − 11.500) 4.750 = 3.250 + 1.500 ✓
The additive identity holds for every single observation. Squaring both sides does not preserve equality for a single observation — a cross-product term 2·(between)·(within) remains. That term vanishes only when summed over all observations in a group, because within-group deviations always sum to zero: Σi(xij − xk.) = 0. This is why SST = SSB + SSW holds in total but not term-by-term.
